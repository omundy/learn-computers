Computer Benchmarks

A computer benchmarking program works by running a series of specific tests to measure performance, ultimately giving a score that allows comparison across machines.

Geekbench

Geekbench is a cross-platform benchmark tool that can measure and track your device's CPU and GPU performance. Their website lets you compare your score to other machines (Mac).

Sysbench

sysbench is a command line multi-threaded benchmark tool primarily for MacOS and Linux. It can be used for database, filesystem-level, CPU, memory access, and many other benchmarks.

# install with homebrew
brew install sysbench
# run a cpu test
sysbench cpu run
# run a memory test
sysbench memory run

Phoronix Test Suite

The Phoronix Test Suite is an open source licensed under the GNU GPLv3 free software license. The Phoronix Test Suite can be used for personal or professional purposes free of charge. Patches to the Phoronix Test Suite are welcome via GitHub.

Compare results on https://openbenchmarking.org

# install with homebrew
brew install phoronix-test-suite
# this install into
which phoronix-test-suite
